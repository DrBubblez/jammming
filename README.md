# Jammming

A web application that lets users search the Spotify library, create custom playlists, and save them to their Spotify account. I'm proud of this project-it helped me deepen my understanding of React and Web APIs through hands-on development.

## Purpose
This project was built to strengthen my skills in React.js, RESTful APIs, and single-page application development. This also reflects my self-driven motivation to grow as a professional web developer.

## Technologies Used

### Core Technologies
- React
- Vite
- JavaScript/JSX
- TailwindCSS v4

### APIs & External Services
- Spotify Web API

### Development Tools
- ESLint
- Git/GitHub
- npm

## Features

### Search Spotify Library
- Search for songs by title

### Create Custom Playlists
- Create and name custom playlist
- Add songs and save playlists directly to your Spotify account
